# Interactive-calculator
This project is an interactive calculator application built entirely with vanilla HTML, CSS, and JavaScript. It was designed to demonstrate core front-end development concepts including DOM manipulation, event handling, input validation, and responsive UI design — all without relying on any external libraries or frameworks.

# Interactive Calculator

A clean, responsive calculator built with vanilla HTML, CSS, and JavaScript.

## Features

- Basic operations: addition, subtraction, multiplication, division
- Percentage and sign toggle (+/−)
- Input validation with user-friendly error messages
- Division by zero protection
- Max 12-digit input limit
- Duplicate decimal point prevention
- Chained calculations
- Live expression history display
- Full keyboard support

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0–9` | Input digits |
| `+ - * /` | Operators |
| `Enter` or `=` | Calculate |
| `Escape` | Clear all |
| `Backspace` | Delete last digit |

## Tech Stack

- HTML5
- CSS3 (custom properties, grid layout)
- Vanilla JavaScript (no libraries)

## How to Run

Just open `calculator.html` in any browser — no setup needed.
